# README

## Описание

Скрипт `shuffle_fonts.py` создаёт новый TTF-шрифт, случайным образом выбирая начертания глифов из нескольких исходных TTF-шрифтов. Для каждого символа (по коду Unicode), присутствующего во всех исходных шрифтах, скрипт случайным образом берёт глиф из одного из шрифтов и копирует его в результирующий шрифт. При этом выполняется:

* Масштабирование глифов, чтобы они соответствовали UPM (units per em) целевого шрифта.
* Копирование метрик горизонтального и вертикального позиционирования (hmtx, vmtx).
* Обновление метаданных шрифта: название семейства, версия, авторское право, описание (список исходных файлов), организация (устанавливается пустой).
* Генерация отдельного текстового файла со списком всех использованных исходных шрифтов.

Такой подход позволяет создавать уникальные «миксовые» шрифты, объединяющие стили разных гарнитур.

## Установка

1. Убедитесь, что у вас установлен **Python 3.6** или новее.
2. Установите необходимую библиотеку `fontTools`:

```bash
pip install fonttools
```

3. Скачайте скрипт `shuffle_fonts.py` в удобную директорию.

## Использование

Запустите скрипт из командной строки, указав папку с исходными TTF-файлами (минимум 2) и параметры выходного шрифта:

```bash
python shuffle_fonts.py --folder /путь/к/папке --output mixed.ttf
```

Если папка не указана, используется текущая директория. Выходной файл по умолчанию — `mixed_font.ttf`.

### Аргументы командной строки

| Аргумент | Описание | По умолчанию |
|----------|----------|--------------|
| `--folder` | Папка, в которой находятся исходные TTF-шрифты. | `.` (текущая) |
| `--output` | Имя выходного TTF-файла. | `mixed_font.ttf` |
| `--family` | Базовое название семейства шрифта. К нему будет добавлен суффикс (если не отключено). | `Regular` |
| `--version` | Строка версии, которая будет записана в метаданные. | `1` |
| `--copyright` | Текст авторского права. Если не указан, берётся из первого шрифта с добавлением суффикса. | `mit` (пример, но можно задать любой) |
| `--no-suffix` | Флаг: если указан, к названиям и авторским правам не добавляется суффикс ` (Mixed ГГГГ)`. | `False` (суффикс добавляется) |
| `--organization` | **Игнорируется.** Организация всегда устанавливается пустой. | - |

> **Примечание:** Суффикс добавляется к полям **Family Name** (nameID 1) и **Copyright** (nameID 0), чтобы отличить сгенерированный шрифт от исходных.

### Пример

```bash
python shuffle_fonts.py --folder ./my_fonts --output my_mixed_font.ttf --family "MyMix" --version "2.0" --copyright "CC BY-SA 4.0"
```

После выполнения:

* Будет создан файл `my_mixed_font.ttf`.
* Создастся файл `my_mixed_font_sources.txt` со списком исходных шрифтов.
* В метаданных шрифта: название семейства станет `MyMix (Mixed 2026)`, авторские права – `CC BY-SA 4.0 (Mixed 2026)`, описание содержит перечень имён исходных файлов, организация пустая.

## Принцип работы

1. Скрипт находит все TTF-файлы в указанной папке.
2. Загружает их и определяет общий набор символов (Unicode-коды), присутствующих во всех шрифтах.
3. Базовым шрифтом становится первый в списке. Его таблица `glyf` и `hmtx` копируется как основа для нового шрифта.
4. Для каждого общего символа:
   * Случайно выбирается один из исходных шрифтов.
   * Из него извлекается соответствующий глиф и метрики.
   * Глиф масштабируется с учётом разницы UPM между исходным и целевым шрифтом.
   * Масштабированный глиф и метрики записываются в целевой шрифт вместо тех, что были из базового.
5. Обновляются метаданные (name-таблица, head.modified).
6. Шрифт сохраняется в указанный выходной файл, а рядом создаётся текстовый файл со списком источников.

## Требования

* Python 3.6+
* Библиотека [fontTools](https://github.com/fonttools/fonttools)

## Ограничения

* Обрабатываются только TTF-шрифты (TrueType).
* Составные глифы (компоненты) пропускаются (не копируются), так как требуют особой обработки.
* Скрипт не проверяет совместимость глифов по дизайну – смешивание может давать визуально неоднородный результат.

## Лицензия

Скрипт распространяется свободно. Вы можете использовать и модифицировать его по своему усмотрению. Исходные шрифты сохраняют свои лицензии – при использовании этого инструмента убедитесь, что лицензии разрешают создание производных работ.
